# Project2-GA
A Web Application that lets the user keep track of their weight. They can create, update, edit and delete their inputs. 

## Technologies Used 
HTML, CSS, JavaScript, JQuery, MongoDB, Mongoose, ExpressJS, nodeJS, Ejs, Git
